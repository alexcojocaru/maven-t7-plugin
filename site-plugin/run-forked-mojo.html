t7:run-forked

Full name:

com.googlecode.t7mp:maven-t7-plugin:0.9.15-SNAPSHOT:run-forked

Description:

Runs a Tomcat 7 (or 6) instance in an separate process (jvm), other than the maven build.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
